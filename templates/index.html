What is Marathon?

Marathon is an online editor and Linux machine for developers who need a quick and easy way to run their code. Built on Docker, it has the ability to run thousands of languages and access all the software available for Linux, all at a blazing-fast speed.

Features

  • 100% Free - Marathon is free and open source, and anyone is free to use and modify it. There is no "pro plan" - all of the features are available to anyone.
  • Fast - Marathon runs code at near-native speeds, allowing you to develop and test rapidly.
  • Powerful - Marathon can access thousands of programming languages and distros through the power of Docker. This allows you to use any number of softwares and languages to write your code.

Getting Started

Marathon has no concept of a 'user'. Instead, individual projects have their own passwords. To get started, simply click on 'Create Project' above, and put in your project name, a Docker image to create it with, and a password. Then you can start developing!

Why the name?

It is called Marathon because it runs you code, just like someone in a marathon does. See what I did there? Puns are great, aren't they?
